머신러닝을 공부하다 보면 **회귀 분석(Linear Regression)** 이나 **주성분 분석(PCA)** 을 학습하는 과정에서 **다중공선성(Multicollinearity)** 과 **특이값 분해(SVD, Singular Value Decomposition)** 라는 개념을 자주 접하게 된다.

처음에는 두 개념이 서로 다른 내용이라고 생각했지만, 공부를 진행하면서 **다중공선성 문제를 이해하려면 SVD의 역할도 함께 이해해야 한다**는 것을 알게 되었다.

이번 글에서는 다중공선성이 왜 문제가 되는지, 그리고 SVD가 데이터를 어떤 방식으로 분석하여 이러한 문제를 완화하는 데 활용되는지 정리해보았다.

---

# 📌 다중공선성(Multicollinearity)이란?

다중공선성이란 **독립 변수들 사이의 상관관계가 매우 높아 서로 비슷한 정보를 가지고 있는 상태**를 의미한다.

쉽게 말해 모델 입장에서는 같은 정보를 여러 번 전달받는 것과 비슷한 상황이다.

---

## 💡 예시 1. 아파트 가격 예측

아파트 가격을 예측한다고 가정해보자.

|평수|방 개수|가격|
|---:|---:|---:|
|20|1|2억|
|30|2|3억|
|40|3|4억|
|50|4|5억|

평수가 증가하면 방 개수도 거의 일정하게 증가한다.

즉,

- 평수
- 방 개수

두 변수는 거의 같은 정보를 담고 있다.

---

## 💡 예시 2. 키를 두 가지 단위로 저장한 경우

|키(cm)|키(m)|
|---:|---:|
|170|1.70|
|180|1.80|
|190|1.90|

두 변수는 단위만 다를 뿐 실제로는 동일한 정보이다.

이처럼 서로 거의 동일한 정보를 가진 변수가 함께 존재하는 것이 다중공선성의 대표적인 예이다.

---

# 🤔 왜 문제가 될까?

예를 들어 회귀 모델이 아래와 같이 집값을 예측한다고 생각해보자.

```text
집값 = 평수 × ? + 방 개수 × ?
```

하지만 평수와 방 개수가 거의 같은 정보를 가지고 있다면 컴퓨터는

> "집값이 오른 이유가 평수 때문일까?"

또는

> "방 개수 때문일까?"

를 명확하게 구분하기 어렵다.

그 결과

- 회귀 계수가 크게 변동할 수 있고
- 모델의 해석이 어려워지며
- 예측 성능이 불안정해질 수 있다.

---

# 📐 기하학적으로 보면

변수가 두 개라면 데이터는 원래 2차원 공간에 분포해야 한다.

```
y

│         ●
│      ●
│   ●
│ ●
└──────────────── x
```

하지만 두 변수가 거의 같은 정보를 가지고 있다면 데이터는 하나의 직선 근처에만 모이게 된다.

즉,

**2차원 공간처럼 보이지만 실제 정보는 거의 1차원에 가깝다.**

이처럼 데이터가 특정 방향으로만 분포하는 현상이 다중공선성이다.

---

# 🔍 SVD(Singular Value Decomposition)란?

SVD는 복잡한 데이터 행렬을

- 새로운 기준(축)
- 각 축의 중요도
- 새로운 좌표

로 분해하는 선형대수 기법이다.

수식으로 표현하면 다음과 같다.

$$
A = U \Sigma V^T
$$

처음에는 공식을 외우려고 했지만, 각 행렬이 어떤 역할을 하는지 이해하니 훨씬 기억하기 쉬웠다.

---

# 📖 각 행렬의 의미

|행렬|역할|
|---|---|
|$V^T$|기존 변수를 조합하여 만든 새로운 기준 축|
|$\Sigma$|각 축이 가지고 있는 정보량(중요도)|
|$U$|새로운 기준에서 각 데이터가 위치한 좌표|

쉽게 말하면

- **Vᵀ는 새로운 좌표축을 만들고**
- **Σ는 각 축이 얼마나 중요한지 알려주며**
- **U는 데이터를 새로운 좌표계로 표현한 결과**

라고 이해하면 된다.

---

# 🎯 SVD는 다중공선성을 어떻게 분석할까?

SVD를 수행하면 가운데 있는 **Σ(Sigma)** 행렬을 확인하게 된다.

Σ에는 각 축이 가지고 있는 정보량이 큰 순서대로 저장된다.

예를 들어 다음과 같은 특이값을 얻었다고 가정해보자.

```text
Σ

[8.3666      0]

[0       7.3e-16]
```

첫 번째 축은 충분한 정보를 가지고 있지만,

두 번째 축은

```
0.0000000000000007
```

수준으로 거의 0에 가깝다.

이는

> "이 방향에는 새로운 정보가 거의 없구나."

라는 의미이다.

즉, 뒤쪽 축은 원래 변수들이 중복되어 만들어진 방향이라고 해석할 수 있다.

실제로 PCA와 같은 차원 축소 기법에서는 이러한 **정보량이 거의 없는 축을 제거**하여

- 중복 정보 감소
- 차원 축소
- 다중공선성 완화

와 같은 효과를 얻는다.

> 💡 참고로 **SVD 자체가 다중공선성을 해결하는 알고리즘은 아니다.**  
> SVD는 데이터를 분해하여 정보량을 분석하는 기법이며, PCA와 같은 알고리즘이 그 결과를 활용하여 중복된 정보를 제거한다.

---

# 💻 Python으로 직접 확인해보기

이번에는 NumPy를 이용하여 SVD를 직접 수행해보자.

아래 데이터는 **두 번째 열이 첫 번째 열의 2배**이므로 완벽한 다중공선성이 존재하는 예제이다.

```python
import numpy as np

# 완벽한 다중공선성이 존재하는 데이터
A = np.array([
    [1, 2],
    [2, 4],
    [3, 6]
])

U, Sigma, VT = np.linalg.svd(A)

print("U")
print(U)

print("\nSingular Values")
print(Sigma)

print("\nV^T")
print(VT)
```

### 실행 결과 (Singular Values)

```text
[8.36660027e+00 7.32018325e-16]
```

SVD를 수행하면 가운데 행렬인 **Σ(Sigma)** 의 대각 성분(특이값, Singular Values)을 얻을 수 있다.

여기서 가장 중요한 점은 **두 번째 특이값이 거의 0에 가깝다**는 것이다.

```text
7.32018325e-16
≈ 0
```

이는 두 번째 방향에는 새로운 정보가 거의 존재하지 않는다는 의미이다.

즉,

- 첫 번째 축에는 대부분의 정보가 존재하고
- 두 번째 축은 기존 변수의 중복으로 인해 생성된 방향이라고 해석할 수 있다.

이처럼 SVD는 데이터의 **정보량이 거의 없는 방향**을 수학적으로 찾아낼 수 있다.

---

# 📊 matrix_rank()도 내부적으로 SVD를 사용한다

NumPy에서 행렬의 랭크(Rank)를 계산하는

```python
np.linalg.matrix_rank()
```

함수 역시 내부적으로 SVD를 이용하여 계산한다.

```python
import numpy as np

A = np.array([
    [1, 2],
    [2, 4],
    [3, 6]
])

print(np.linalg.matrix_rank(A))
```

### 실행 결과

```text
1
```

겉으로 보기에는 열(Column)이 2개이므로 독립적인 정보도 2개처럼 보일 수 있다.

하지만 실제로는

```text
두 번째 열 = 첫 번째 열 × 2
```

이므로 새로운 정보는 하나뿐이다.

따라서 행렬의 Rank는 **1**이 된다.

> 💡 Rank는 "서로 독립적인 정보(축)의 개수"라고 이해하면 기억하기 쉽다.

---

# 🤖 머신러닝에서는 왜 중요할까?

SVD는 단순히 행렬을 분해하기 위한 수학 공식이 아니라 **데이터의 구조를 분석하는 핵심 도구**이다.

대표적으로 다음과 같은 분야에서 활용된다.

|활용 분야|SVD의 역할|
|---|---|
|선형회귀(Linear Regression)|다중공선성 분석 및 수치적 안정성 향상|
|PCA(주성분 분석)|정보량이 큰 방향을 찾아 차원 축소|
|추천 시스템|사용자-아이템 행렬 분해(Matrix Factorization)|
|자연어 처리(LSA)|문서의 잠재 의미(Latent Semantic) 추출|
|이미지 압축|정보 손실을 최소화하면서 데이터 압축|
|노이즈 제거|정보량이 작은 특이값을 제거하여 잡음 감소|

예를 들어 **PCA**에서는 SVD를 이용해 데이터의 분산이 큰 방향을 찾고, 정보량이 거의 없는 축은 제거하여 데이터를 더 적은 차원으로 표현한다.

또한 추천 시스템에서는 사용자와 상품의 관계를 작은 행렬로 분해하여 사용자의 숨겨진 선호도를 학습하는 데 활용된다.

즉, SVD는 다양한 머신러닝 알고리즘의 기반이 되는 매우 중요한 선형대수 기법이다.

---

# 📝 정리

이번 내용을 공부하면서 가장 인상 깊었던 점은 **SVD가 단순히 행렬을 분해하는 공식이 아니라 데이터가 가지고 있는 정보의 구조를 분석하는 방법**이라는 점이었다.

처음에는 다중공선성과 SVD가 서로 다른 개념이라고 생각했지만, SVD를 통해 데이터가 어떤 방향으로 얼마나 많은 정보를 가지고 있는지 분석할 수 있고, 이러한 결과가 PCA와 같은 차원 축소 기법에서 활용된다는 점을 이해할 수 있었다.

또한 Python으로 직접 SVD를 수행하고 특이값(Singular Values)을 확인해보면서, 정보량이 거의 없는 방향이 실제로 존재한다는 것도 확인할 수 있었다.

앞으로 PCA와 추천 시스템 등 SVD가 활용되는 다양한 머신러닝 알고리즘을 학습하면서도 **"왜 이런 연산이 필요한가?"**, **"어떤 문제를 해결하기 위해 사용하는가?"**를 함께 이해하는 방향으로 공부해볼 필요가 있을 것 같다.
