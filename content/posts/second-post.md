## 🏗️ 휴고(Hugo) 블로그 구조 한눈에 보기

| 디렉토리/파일     | 아이콘 | 역할                           | 관리 난이도   |
| ----------- | --- | ---------------------------- | -------- |
| `content/`  | ✍️  | 당신의 생각과 글이 저장되는 공간 (블로그의 본체) | ⭐        |
| `themes/`   | 🎨  | 블로그 디자인과 레이아웃을 담당하는 테마       | ⭐⭐       |
| `data/`     | 📊  | 공통 데이터 및 설정 정보 저장            | ⭐⭐⭐      |
| `static/`   | 🖼️ | 이미지, 아이콘, 파일 등의 원본 저장소       | ⭐        |
| `public/`   | 🌐  | 최종 생성된 HTML 웹사이트             | 🔒 자동 생성 |
| `hugo.toml` | ⚙️  | 블로그 전체 설정을 담당하는 핵심 파일        | ⭐⭐⭐      |

### 📌 한 줄 설명

* **`content/`** → 블로그 글을 작성하는 가장 중요한 폴더
* **`themes/`** → 블로그의 외형(디자인)을 결정
* **`data/`** → 사이트 전반에서 사용하는 데이터 저장
* **`static/`** → 이미지, PDF, favicon 등을 보관
* **`public/`** → Hugo가 자동으로 생성하는 결과물
* **`hugo.toml`** → 사이트 제목, 주소, 테마 등을 설정
